# Мониторинг новостей AI

Проект для отслеживания и анализа новостей в сфере искусственного интеллекта.

## Структура проекта

- `articles/` - сохраненные статьи и новости
- `scripts/` - скрипты для мониторинга и обработки
- `data/` - данные и аналитика
- `reports/` - отчеты и аналитические материалы

## Цели проекта

- Автоматический мониторинг новостей AI
- Систематизация информации
- Анализ трендов в области AI

## Начало работы

Все материалы по мониторингу новостей AI сохраняются в соответствующие папки этого репозитория.
